# Merk

This is experimental code for turning arbitrary data into a Merkle tree and comparing two trees.

## Installation

(the usual methods)
Add this line to your application's Gemfile:

```ruby
gem 'merk'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install merk
